   1. Umowa dotyczy lokalu o powierzchni {frontmatter.area} m<sup>2</sup>, znajdującego się pod adresem: **{frontmatter.address} / {frontmatter.apt}**.
   1. W skład lokalu wchodzą następujące pomieszczenia:
      <ul>
      {Number(frontmatter.room) > 0 && (<li>{frontmatter.room} x pokój</li>)}
      {Number(frontmatter.kitchen) > 0 && (<li>{frontmatter.kitchen} x kuchnia</li>)}
      {Number(frontmatter.hall) > 0 && (<li>{frontmatter.hall} x pokprzedpokójój</li>)}

      {Number(frontmatter.balcony) > 0 && (<li>{frontmatter.balcony} x balkon</li>)}
      {Number(frontmatter.wardrobe) > 0 && (<li>{frontmatter.wardrobe} x garderoba</li>)}
      {Number(frontmatter.garage) > 0 && (<li>{frontmatter.garage} x garaż</li>)}
      </ul>
      {frontmatter.systems !== 'brak' && (<li>W lokalu dostępne są instalacje: {frontmatter.systems}</li>)}

      <ul>

   1. Zwrot lokalu powinien nastąpic do {frontmatter.return}. dnia od daty zakończenia Umowy.
